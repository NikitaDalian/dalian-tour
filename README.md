# 🏴‍☠️ Тайны старого Даляня

Интерактивный пешеходный квиз-маршрут по историческому центру Даляня для детей 8–14 лет.

## Что это

Веб-приложение с интерактивной картой (Gaode/高德) и квизом на 12 остановок:
- Железнодорожный вокзал → Площадь Победы → Площадь Чжуншань и банки → Финиш
- На каждой остановке: история, задание «найди» и вопрос с вариантами ответов
- Система очков, достижения, диплом в финале

## Запуск

Просто откройте `index.html` в браузере. Никаких зависимостей, сборки или сервера не нужно.

Или разверните через GitHub Pages — см. ниже.

## GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/dalian-tour.git
   git push -u origin main
   ```
3. Перейдите в **Settings → Pages → Source: main / root → Save**
4. Через 1–2 минуты сайт будет доступен по адресу:
   `https://YOUR_USERNAME.github.io/dalian-tour/`

## Карта

Используются тайлы **Gaode (高德地图)** — работают в Китае без VPN и без API-ключа.
Координаты автоматически конвертируются из WGS-84 в GCJ-02.

## Технологии

- Leaflet.js (карта)
- Gaode/Amap tiles (тайлы)
- Vanilla JS (без фреймворков)
- Всё в одном HTML-файле (~100KB, включая встроенные изображения)
